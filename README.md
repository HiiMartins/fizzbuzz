# FizzBuzz
## Projeto do curso Elixir e Phoenix do Zero!

Esse projeto tem como objetivo ler um arquivo que contém sequência numérica, transformar essa sequência que virá no formato de string em uma lista de numbers e verificar a lista, afim de substituir os multiplos de 3 por *:fizz*, os multiplos de 5 por *:buzz* e os multiplos de ambos, por *:fizzbuzz*.

### Conceitos Praticados no projeto:

* [Atom](https://elixir-lang.org/getting-started/basic-types.html#atoms)
* [Enum/Enumerables](https://elixir-lang.org/getting-started/enumerables-and-streams.html)
* **Pipe Operator**
* [Pattern Matching](https://elixir-lang.org/getting-started/pattern-matching.html)
* **Guards**
